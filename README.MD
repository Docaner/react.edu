## Вопросы
- Список новостей/комментариев оформить в pages или в widgets?
- Features/addNews - есть кнопка создания страницы на домашней странице и есть кнопка создания на странице добавления новости. Одна кнопка перенаправляет на страницу, а другая - создаёт новость и отправляет пользователя на новую страницу. Эту фичу можно объеденить в одну кнопку или лучше реализовать в 2 разные фичи?
